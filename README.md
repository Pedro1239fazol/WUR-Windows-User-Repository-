# WUR - Windows User Repository

## Sobre o projeto

O **WUR (Windows User Repository)** é um gerenciador de programas para Windows desenvolvido em Python.

Ele permite que o usuário escolha um programa, selecione uma versão disponível e faça o download do instalador automaticamente através de um repositório de links.

## Recursos

* Lista de programas disponíveis.
* Seleção de versões.
* Download automático dos instaladores.
* Execução automática do setup após o download.
* Sistema simples de repositório baseado em Python.

## Como funciona

1. O WUR mostra os programas disponíveis.
2. O usuário escolhe um programa.
3. O WUR mostra as versões disponíveis.
4. O usuário escolhe a versão.
5. O instalador é baixado.
6. O instalador é executado automaticamente.

##Como é composto

O WUR é composto por muitos arquivos mas é muito leve e também é de Codigo Aberto, Sim de Codigo Aberto ou seja você pode baixar o codigo fonte em https://github.com/Pedro1239fazol/WUR-Windows-User-Repository-
Lembre o WUR não foi feito para ser customizado por terceiros e também tem sempre atualizações ente 1 a 7 dias, sempre o atualize para ter versões novas e novos APPS viste o site no SourceForge: https://sourceforge.net/projects/windows-user-repository/ ou no site do GitHub: https://github.com/Pedro1239fazol/WUR-Windows-User-Repository- 

## Objetivo

O objetivo do WUR é criar um sistema simples de distribuição de programas para Windows, semelhante a um gerenciador de pacotes.
